API
======

API of super secret project

##Install
`npm install`

`git submodule init, sync`

`brew install protobuf`

`sh bootstrap.sh`

#### CREATE

localhost:8001/create?address={address}&amount={amount}&memo={memo}&ackmemo={ackmemo}

#### PAY

localhost:8001/pay/{id}

#### GET

localhost:8001/get/{id}
